# app-9games-guide

## 项目简介

本仓库用于归档与发布多个独立的 HTML 页面。这些页面以静态网页形式存储，不针对任何特定域名或网站，主要作为通用 HTML 资料集合使用。

## 目录说明

```
app-9games-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（图片、样式、脚本等）
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

- **pages/**：每个 HTML 文件为一个独立页面，可直接在浏览器中打开。
- **assets/**：按类型或页面分类存放资源，便于管理。
- **docs/**：可选的补充文档，如页面功能说明或使用指引。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖后端服务。
- 页面之间相互独立，无强耦合关系。
- 归档内容可能随时间更新或调整，建议以仓库最新版本为准。
- 页面可能引用外部资源（如 CDN 库），请确保网络环境允许访问。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 以修正页面错误或提出改进建议。
- 新增页面时，请遵循以下约定：
  - 页面文件放置于 `pages/` 目录下。
  - 资源文件放置于 `assets/` 下对应子目录。
  - 保持页面结构简洁，避免冗余依赖。
- 仓库维护者将不定期检查并合并合理贡献。
